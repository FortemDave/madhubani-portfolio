# Madhubani Art Portfolio

A modern, responsive portfolio showcasing traditional Madhubani artwork by Ritu Dave.

## Features
- Dynamic diagonal hero animation
- Lazy-loaded WebP images for optimal performance
- Glass morphism design
- Responsive layouts for all devices
- Zoom functionality for artwork details

## Setup
1. Clone the repository
2. Enable GitHub Pages in Settings > Pages
3. Select branch: `main` and folder: `/ (root)`
4. Site will be live at `https://yourusername.github.io/repository-name/`

## Custom Domain (Optional)
1. Add your domain to the `CNAME` file
2. Configure DNS with your domain provider
3. Enable HTTPS in GitHub Pages settings

## Performance
- All images optimized as WebP format
- Lazy loading enabled for below-the-fold content
- Critical resources preloaded
- Proper caching headers via GitHub Pages CDN
